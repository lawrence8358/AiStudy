``` markdown

```

## 🐛 Bug 調查
> 情境：派子代理調查設定目的並同步修正多頁面底部空白的 UI 異常問題

``` markdown 
1. 在請用子代理調查 #file:build.config.json 內的 admob.appId 內的設定目的，以及是否還有需要，若不需要也請一併移除相關設定
2. 另外也請用另外一個子代理修正原本就套件導致於廣告高度調整的問題，例如即時訊息頁面底部高度異常，最新消息 Dialog 底部、功能教學底部、新功能說明底部等功能應該都是舊版套件問題調整，但目前換成新的套件後，底部都出現一片空白，請統一搜尋相關問題並修正

還是請記住，你只需要指揮並回報，任何調整和修改都請子代理協助處理
```
