你是一位資深 AI 工程師，正在協助一位 .NET 背景的面試官準備 AI 工程師職缺的面試。
面試官本身不具備 AI 開發技能，因此需要你提供完整的面試指引與考題。

---

技術棧以 .NET Core 為主，主要方向為 AI 落地應用，例如：
- 會議記錄語音轉文字，結合 LLM 與語者分離技術，產生逐字稿與會議重點
- 客製化 LLM + RAG 系統，針對客戶文件建置專屬問答系統
- 雲端與地端模型串接整合

# 任務要求

請依照底下格式，產生一份 Markdown 面試指引文件，包含以下三個區塊：

## 1. 能力分級說明
說明如何根據回答品質，判斷面試者屬於哪個等級：
- **初階（Junior）**：能使用現成工具，但理解有限
- **中階（Mid）**：能獨立完成功能，理解原理
- **高階（Senior）**：能設計架構、優化系統、解決複雜問題

## 2. 面試問題清單
依照核心能力領域分組，每組提供：
- 破冰暖身題（1 題，確認基本背景）
- 核心評估題（2～3 題，區分中高階）
- 追問題（每題至少 1 個追問，用於深入判斷）

涵蓋領域：
1. LLM API 串接與 Prompt Engineering
2. RAG 系統設計與實作
3. Python 開發與 API 部署
4. 影像識別應用
5. 客戶溝通與需求轉化

## 3. 考題（選擇題 + 實作題）
- 選擇題：5 題，附正確答案與各選項說明，供面試官快速評估基礎觀念
- 實作題：2 題，提供題目描述、預期輸出，以及評分重點（面試官不需親自驗證程式）

---

# 輸出格式要求

- 使用 Markdown 格式
- 每個問題加上「**初階可答**」/ 「**中階可答**」/ 「**高階可答**」標籤
- 考題附上「參考答案」或「評分重點」，讓不具備 AI 背景的面試官也能判讀結果
